# Expense Tracker Project

This project analyzes personal expenses using Python.

## Tools
- Python
- Jupyter Notebook
- pandas
- matplotlib

## Files
- expense_tracker.ipynb – Project notebook
- expenses.csv – Dataset
- Expense_Tracker_Project_Report.pdf – Project report

## Features
- Total expense calculation
- Category-wise expense analysis
- Pie chart and bar chart visualization

## Conclusion
This project demonstrates how Python can be used for simple expense analysis and data visualization.
